## üöÄ How to Use

### Voice Commands
1. Open the frontend URL
2. Click the "Speak" button
3. Allow microphone permissions
4. Say commands like:
   - "Add milk"
   - "Add 2 bread"
   - "Show my list"
   - "Remove banana"
   - "Search for toothpaste"

### Text Input
1. Type commands in the text field
2. Press Enter or click "Send"
3. Same commands work as voice
